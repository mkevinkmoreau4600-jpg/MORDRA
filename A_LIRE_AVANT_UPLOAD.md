# MORDRA — DOSSIER À METTRE SUR GITHUB

Ce dossier contient la version prête à publier sur GitHub Pages.

## Ce qu'il faut envoyer dans ton dépôt MORDRA

Envoie tous ces fichiers/dossiers à la racine du dépôt :

- index.html
- style.css
- game.js
- manifest.webmanifest
- sw.js
- icons/
- README.md

Tu peux aussi envoyer :
- INSTALLER_IPHONE.html

## Important

Ne mets pas le fichier ZIP directement sur GitHub.
Il faut ouvrir le ZIP puis envoyer les fichiers à l'intérieur.

## Étapes GitHub

1. Va dans ton dépôt MORDRA.
2. Clique sur "Ajouter un fichier".
3. Clique sur "Téléverser des fichiers".
4. Glisse tous les fichiers du dossier MORDRA_GITHUB_READY.
5. Clique sur "Valider les modifications".
6. Attends 1 minute.
7. Ouvre :
   https://mkevinkmoreau4600-jpg.github.io/MORDRA/

## iPhone

Sur Safari :
1. Ouvre le lien du jeu.
2. Appuie sur Partager.
3. Appuie sur "Sur l'écran d'accueil".
4. Appuie sur Ajouter.
