# MORDRA v0.3.0 — The Hunt Update

Première grosse mise à jour du prototype.

## Nouveautés principales
- Nouveau moteur de partie.
- Joueurs vivants / éliminés.
- Plusieurs Tueurs gérés correctement.
- Si un Tueur est éliminé mais qu'il en reste un, la partie continue.
- Les mots restent les mêmes pendant toute la manche.
- Les joueurs éliminés ne reçoivent plus le téléphone.
- Les joueurs éliminés ne votent plus.
- Le vote ne propose que les joueurs vivants.
- Même son de révélation pour Survivant et Tueur.
- Plus aucun son qui permet de deviner le rôle.
- Conditions de victoire automatiques.
- Égalité : nouveau tour, pas d'élimination au hasard.
- Après 3 égalités : victoire des Tueurs.
- Historique de partie avec journal interne.
- XP, stats, jauges et passages de niveau conservés.

## Utilisation
Ouvre `index.html` dans un navigateur.
